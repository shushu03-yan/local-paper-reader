# Examples

This folder contains a small public demo set for the GitHub release package.

Included:

- `demo.pdf`
- `demo_output/paper.md`
- `demo_output/reader.md`
- `demo_output/manifest.json`

Intentionally excluded:

- full historical output directories
- SQLite databases
- logs
- local absolute paths
- API keys or secrets
- page image assets and other bulky output files not required to understand the workflow
